# Cover Generator

## Trigger when

- 用户说“生成封面”
- 用户说“帮我配一张封面图”
- 用户说“做个头图”
- 用户说“出三版封面”
- 用户说“给文章配图”

## Goal

基于 baoyu-cover-image 能力封装，根据标题、平台和人设风格生成 3 个封面备选。

## Platform ratios

| 平台 | 比例 | 风格建议 |
| --- | --- | --- |
| 小红书 | 1:1 | 鲜艳、高对比、强标题感 |
| 微信公众号 | 2.35:1 | 专业、留白、品牌感 |
| 知乎 | 16:9 | 简洁、大气、信息型 |
| CSDN | 16:9 | 技术感、深色、结构化 |
| 今日头条 | 16:9 | 资讯感、标题突出 |
| 掘金 | 16:9 | 技术与工具感 |

## Rules

- 封面文案必须来自文章标题核心信息，而不是重新发散。
- 风格参数优先读取人设中的 `writing_style` 与 `personality`。
- 输出 3 个候选方向，分别强调：稳妥版、传播版、品牌版。

## Output

- 每个平台对应的尺寸
- 3 个封面文案方案
- 3 条封面生成提示
