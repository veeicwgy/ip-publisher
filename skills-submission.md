# IP Publisher

## 一句话描述

面向个人 IP 运营者的全流程内容自动化 Skill，从热点抓取到多平台发布一条链路完成。

## 功能亮点

1. 一句话触发完整内容生产流程。
2. 先做人设定义，再做热点筛选，减少内容跑偏。
3. 内置多平台写作规范，支持小红书、知乎、公众号、CSDN、微博、头条、掘金。
4. 集成中文去 AI 味能力，强化个人表达风格。
5. 支持封面生成与多平台草稿/直发发布。

## 安装方法

1. `git clone https://github.com/veeicwgy/ip-publisher.git && cd ip-publisher`
2. `bash scripts/setup.sh`

## 示例触发词

- 帮我写一篇关于 AI 趋势的小红书文章
- 基于今天的热点写一篇公众号推文
- 设置我的 IP 人设
- 给我抓取今天适合我的热点
- 一键发布到知乎和公众号
