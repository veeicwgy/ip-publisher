# IP Publisher Skills

这是一个面向个人 IP 运营者的全流程内容自动化 Skill 集合。

## Skills

- `ip-publisher` - 主流程 Skill，一句话触发完整 IP 内容发布流程
- `ip-profile` - 定义和保存你的 IP 人设
- `hotspot-fetcher` - 抓取多平台实时热点
- `article-generator` - 基于人设 + 热点生成指定平台文章
- `humanizer` - 消除文章 AI 生成痕迹
- `cover-generator` - 生成文章封面图
- `multi-publisher` - 多平台一键发布
- `github-open-source-site-rework` - 将官网从收费导向改为 GitHub 开源仓库导向

## 快速开始

说[帮我写一篇小红书文章]即可触发完整流程。
